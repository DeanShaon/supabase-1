ilike()用于查找所有在所述`列（column）`上的值与提供的 `模板（pattern）`相符的记录（不区分大小写）。





## 案例教程
### 案例1 (使用select)

<Tabs
  scrollable
  size="small"
  type="underlined"
  defaultActiveId="demo2"
>

<TabPanel id="demo2" label="使用方法">

```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .ilike('name', '%la%');
```
</TabPanel>
</Tabs>

<Tabs
  scrollable
