你可以用"远程程序调用"的方式调用存储程序。

这是一种高级的说法，即你可以把一些逻辑放入数据库，然后从任何地方调用它。
这在逻辑很少更改的情况下特别有用，比如密码重置和更新等情况。




## 案例教程
