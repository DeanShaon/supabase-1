neq()用于匹配`列`值不等于`指定值`的行。




## 案例教程
### 案例1 (使用select)

<Tabs
  scrollable
  size="small"
  type="underlined"
  defaultActiveId="demo2"
>

<TabPanel id="demo2" label="使用方法">

```dart
                                                                              
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .neq('name', 'The shire');
```
</TabPanel>
</Tabs>
