match()用于查找表（table）中所有列与指定的`查询（query）`对象相匹配的行。



## 案例教程
### 案例1 (使用select)

<Tabs
  scrollable
  size="small"
  type="underlined"
  defaultActiveId="demo2"
>

<TabPanel id="demo2" label="使用方法">

```dart
                                                                              
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .match({'name': 'Beijing', 'country_id': 156});
```
</TabPanel>
</Tabs>
