from.createSignedUploadUrl()用于创建一个带有签名的上传URL。

带有签名的上传URL可用于在不需要额外认证的情况下将文件上传到存储桶中，这些URL的有效期为2小时。


需要RLS策略权限:
  - `buckets`表的权限: 无
  - `objects`表的权限：`insert`

请参考[存储指南](/docs/guides/storage#access-control)中关于访问控制的工作方式。
