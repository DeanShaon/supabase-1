resend()用于重新发送验证码
重新发送已存在的注册确认邮件、更改邮箱邮件、短信验证码或更改手机号验证码。

* 将注册确认邮件、更改邮箱或更改手机号的邮件重新发送给用户。
* 无密码登录可以通过再次调用 `signInWithOtp()` 方法进行重新发送。
* 密码找回邮件可以通过再次调用 `resetPassswordForEmail()` 方法进行重新发送。
* 仅当最初进行过注册、更改邮箱或更改手机号的请求时，此方法才会向用户重新发送邮件或短信验证码。
